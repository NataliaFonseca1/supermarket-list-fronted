## Running Instructions

1. Clone the project:
```
https://github.com/NataliaFonseca1/supermarket-list-fronted.git
```

2. Install the dependecies:
```
cd supermarket-list-frontend && npm install
```

3. Run the project:
```
npm start
```
4. (Optional) If you want build a production version:
```
npm build
```
